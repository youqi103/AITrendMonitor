# AI Trend Monitor — 需求文档

## 1. 项目背景

用户是一名炒股短线玩家，需要第一时间获取股市热点信息（如股票利好、暴雷事件等）。传统人工搜索方式效率低、时效性差，需要自动化工具来实时监控和发现热点变化，并及时推送通知。

## 2. 核心需求

### 2.1 关键词监控

- **输入**：用户手动输入要监控的关键词（如"特斯拉"、"美伊冲突"、"降息"等）
- **匹配**：定时爬取财经新闻，利用 AI 判断内容是否与关键词真正相关
- **防伪**：利用 AI 交叉验证多源信息，识别并过滤假冒/虚假内容
- **通知**：匹配成功后第一时间通过 Socket.IO 实时推送 + 浏览器 Notification 通知

### 2.2 热点发现

- **自动搜集**：每隔一段时间，自动爬取指定财经数据源的最新资讯
- **范围限定**：用户可指定关注范围（如"地缘政治"、"新能源"、"AI 概念"等）
- **AI 提取**：利用 AI 自动从海量资讯中提取热点主题
- **排序展示**：按热度/时间排序，让用户快速了解当前市场焦点

### 2.3 Agent Skills 技能接口

- 将核心功能封装为标准化 REST API
- 其他 AI Agent 可调用这些 API 来执行监控和发现任务
- 技能包括：`monitor_keyword`、`discover_trends`、`get_alerts`

## 3. 产品形态

### 3.1 Web 页面（优先开发）

- 响应式兼容的 Web 页面，桌面端和移动端均可使用
- 赛博交易终端风格：深色底 + 霓虹绿/红涨跌色 + 数据流滚动效果
- 实时数据推送，无需手动刷新

### 3.2 Agent Skills（后续开发）

- 提供 OpenAPI 规范的 REST API
- 可被其他 AI 系统集成调用

## 4. 数据源

| 数据源 | 类型 | 说明 |
|--------|------|------|
| 东方财富 | 财经门户 | A 股资讯最全，7x24 小时快讯 |
| 新浪财经 | 财经门户 | 覆盖面广，新闻更新快 |
| 同花顺 | 券商平台 | 短线数据丰富，概念板块全 |
| 雪球 | 投资社区 | 社区讨论多，适合发现民间热点 |

## 5. 非功能需求

| 需求 | 说明 |
|------|------|
| 实时性 | 关键词匹配后 30 秒内推送通知 |
| 准确性 | AI 识别假内容准确率 > 85% |
| 轻量性 | 单机部署，零外部依赖（除 AI API 外） |
| 易用性 | 开箱即用，配置简单 |
| 响应式 | 桌面/平板/手机均可正常使用 |

## 6. 用户角色

| 角色 | 说明 |
|------|------|
| 普通用户 | 通过 Web 页面操作，添加关键词、查看热点、接收通知 |
| AI Agent | 通过 Skills API 调用，自动执行监控和发现任务 |

## 7. 开发优先级

1. ✅ Web 页面 + 后端 API（核心功能）
2. ✅ AI 内容识别与热点分析
3. ⬜ Agent Skills 技能接口（Web 版稳定后开发）
